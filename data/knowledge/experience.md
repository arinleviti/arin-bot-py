# Technical Experience

## Angular
Arin has built production applications with Angular, including his own portfolio website (https://arinleviti.site) and a web frontend for a space strategy game inspired by Ogame, developed for an independent French studio. He is comfortable with Angular's component-based architecture, two-way data binding, routing, services, dependency injection, and i18n for multilingual support. The portfolio site's hero section uses Angular i18n for Italian/English translation. He also built the chat widget integrated into this site using Angular.

## React and Next.js

## TypeScript
